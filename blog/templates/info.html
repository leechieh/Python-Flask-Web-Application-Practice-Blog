About

This site is built using python Flask. It is a text only message board type blog site. Each posts can be viewed by anyone in public. Only registered user is able to create new posts, and only the author of the post can edit or delete the post. User is able to delete his or her own account once registered, and once the account is deleted, the posts authored by the account will be deleted as well.



For more information, please check my github page.
